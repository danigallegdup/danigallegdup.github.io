How Does The Internet Work?

What is the Internet
3min
As Physical System
  • • How does digital information travels from one device to another?
  • • Every piece of digital data transferred from the server to the client is sent in binary representation
  • • Electricity: Electricity: Seen telephone lines around town, ethernet sockets at your house growing up, High voltage, Low voltage
  • • Radiowaves: Wifi
  • • Light: Signals transferred through a fibre optic cable that was placed under the ocean
  • • the internet is everything that digital information travels from one device to another
Servers
3min
HTTPS
  • • HyperText Transfer Protocol Secure
  • • Everything on the internet is stored on a server. Servers are a computer that can store a massive amount of data
  • • An internet client (laptop, PC, phone, etc.) sends requests to a server in order to view, update, create or delete anything from that computer.
  • • The primary function of HTTP is to establish a connection between a server and an internet client, and enable the server to send HTML content back to the user.
Connection to Previous Lectures
3min
Summary
  • • Encryption: The 'S' in HTTPS means Secure. This means that there is end-to-end encryption from server to user.
  • • Binary Information: Light, Electricity, Radiowaves.
  • • Logic Gates: The sofware that optimizes the path the packets will take.
